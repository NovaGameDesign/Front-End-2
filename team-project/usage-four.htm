Physics 2D Overview

GSAP's Physics 2D incorporates multiple mathematical functions allowing users to simulate velocity, angle, gravity, acceleration, accelerationAngle, and/or friction. This implementation should not be considered a fully fledged physics engine like matter.js.

Use Cases
