Getting Started With GSAP

GSAP, short for GreenSock Animation Platform, is a powerful JavaScript library for creating high-performance animations for the web. This guide will walk you through the process of installing GSAP and getting started with basic animations.


Installing GSAP

To install GSAP, you have a couple of options:


Option 1: CDN

The fastest and easiest way to get started with GSAP is to use the CDN hosted by them.

html
<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.9.1/gsap.min.js"></script>

Just copy and paste the script tag above into your HTML, and you're ready to use GSAP.


Option 2: NPM

However, if you prefer to use a package manager, you can also install GSAP via npm.

bash
npm install gsap

After that, import GSAP into your JavaScript file.

javascript
import { gsap } from "gsap";

Option 3: Download

You can also download GSAP directly from here.
