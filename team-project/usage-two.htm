ScrollTo Effect:


Using GSAP's ScrollTo Plugin


How to Use GSAP's ScrollTo Plugin

You must access the GSAP library and inculde the plug ins in your HTML. You can do this by either downloading the libraries and linking to them locally or by including them from a Content Delivery Network (CDN).

Once you specify the target destination you want to scroll to, either by providing a CSS selector that matches an element on your page or by specifying the exact position in pixels. When you trigger the scroll animation, GSAP takes care of smoothly transitioning the viewport to the specified destination.



What is a ScrollTo Effect?

The ScrollTo effect allowes you to features navigation links at the top of your page that smoothly scroll to specific sections when clicked.

By using the GSAP ScrollTo plugin, you can enhance the user experience on your website by providing smooth and intuitive navigation between different sections and elements.
