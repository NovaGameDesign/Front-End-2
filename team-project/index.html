Powerful animation with simple code, GreenSock Animation Platform

GSAP, which stands for GreenSock Animation Platform, is a powerful JavaScript animation library used for creating high-performance, complex animations for the web. Using GSAP developers can animate practically anything with simple APIs using JavaScript. With its powerful suite of tools GSAP can be seen in use on a multitude of websites such as YouTube and Amazon.

GSAP Can be simple

GSAP doesn't have to be complex... as some of the best animations be be created with only a few lines of code!

GSAP's Tween feature allows developers to make simple interactions like changing the position of an element or changing its rotation! This functionality can then be extended to be paused, played, reversed, or even restarted from the beginning! Plus, GSAP works with almost anything, not just HTML elements! It is completely possible to take this interaction and apply it to other things like videos.

GSAP's Many plugins

GSAP has not just one or two plugins, but over twenty optional plugins you can use to extend their base functionality.

  1. ScrollTrigger: This plugin allows you to trigger animations based on the scroll position of the page. It's commonly used for creating scroll-based animations such as parallax effects, pinning elements, and triggering animations as they come into view.
  2. SplitText: SplitText enables you to animate individual characters, words, or lines within a text element. This is useful for creating eye-catching text animations, like typewriter effects or text reveals.
  3. DrawSVG: DrawSVG allows you to animate the drawing of SVG paths. This is often used to simulate the drawing of illustrations or diagrams, providing a hand-drawn effect.
  4. MorphSVG: MorphSVG enables you to smoothly morph between different SVG shapes or paths. It's commonly used for creating organic animations, such as transforming one SVG icon into another.
  5. BezierPlugin: BezierPlugin extends GSAP's easing capabilities by allowing you to define custom Bezier curves for tweening animations. This gives you more control over the acceleration and deceleration of animations.
