Powerful animation with simple code, GreenSock Animation Platform

GreenSock Animation platform, or GSAP for short, is a powerful animation library for web development. Using GSAP developers can animate practically anything with simple APIs using JavaScript. With its powerful suite of tools GSAP can be seen in use on a multitude of websites such as YouTube and Amazon.
